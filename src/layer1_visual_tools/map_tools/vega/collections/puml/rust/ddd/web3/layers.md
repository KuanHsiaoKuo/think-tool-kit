| :-------------- | :------------------- | :---- |

|  | 架构层-区块链层 | 节点管理、通信协议、事件调度等 |

|  | 网络层 | 点对点网络、矿工、节点等 |

|  | 应用层 | 去中心化交换协议（DEX）、去中心化金融（DeFi）应用、区块链游戏、溯源应用、存证应用等 |
| Layer3应用层 | 架构层-前端层 | HTML、CSS、JavaScript、MVC等 |
|  | 架构层-后端层 | web框架、库、RPC、RESTful API等 |

|  | 安全层 | 非对称加密算法、数字签名、智能合约安全编写等 |
|  | 网络层 | 点对点网络、矿工、节点等 |
